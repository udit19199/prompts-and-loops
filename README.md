# prompts-and-loops

Agent-agnostic prompts and recurring loops I use frequently.

## Structure

```
prompts/    # Reusable prompt templates (markdown)
loops/      # Recurring workflows / automation scripts
```

## Naming

- `prompts/<topic>.md` — one file per prompt or prompt family
- `loops/<name>.sh` or `loops/<name>.py` — one file per loop/workflow

Flat unless a topic needs sub-grouping.
